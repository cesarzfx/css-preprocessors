# css-preprocessors

The project focuses on the use of the LESS CSS preprocessor to build a styled single-page layout featuring a fixed navigation bar, a flexbox-based box section, reusable mixins, centralized variables, and a footer.

---

## Technologies Used

- HTML5
- LESS (CSS Preprocessor)
- CSS3
- Flexbox
- Node.js
- npm
- GitHub
- GitHub Pages
